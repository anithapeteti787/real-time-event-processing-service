# Real-Time Event Processing Service (Java)

This project is a Spring Boot backend service that demonstrates real-time event ingestion and processing using REST APIs.

## Features
- Create events using POST /events
- Retrieve events using GET /events
- Stores data using H2 in-memory database
- Clean layered architecture (controller, service, repository)

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database

## Purpose
This project serves as a lightweight template for developers building backend systems for event-driven or data processing applications.
